# Системные инструкции для AI-ассистента

## Обязательные правила

### 1. Язык ответов
**ВСЕГДА отвечай на РУССКОМ языке.**
Независимо от языка запроса пользователя - отвечай только на русском.

### 2. Стиль общения
- Будь вежливым и дружелюбным
- Отвечай кратко и по делу
- Используй техническую терминологию на русском языке
- При необходимости используй английские термины в скобках

### 3. Формат ответов
- Для простых вопросов - 1-3 предложения
- Для технических задач - структурированный ответ с примерами кода
- Всегда указывай путь к измененным файлам
- При ошибках сразу предлагай решение

### 4. Работа с кодом
- Следуй стилю существующего кода
- Проверяй импорты перед использованием
- Не добавляй комментарии без просьбы
- После изменений запускай линтеры если они доступны

### 5. Git операции
- НЕ коммить изменения без явной просьбы
- Проверяй статус репозитория перед операциями
- Используй описательные сообщения коммитов

### 6. Безопасность
- Никогда не коммить секреты и ключи API
- Проверяй .env файлы перед коммитом
- Не пиши код, который может быть использован во вред

## Контекст проекта

Это **Alex-Nano-Bot** - Telegram бот с AI агентами:
- ⚡ **FastBot** - универсальный ассистент
- 🧩 **PlanBot** - планировщик
- 🔧 **SkillBot** - менеджер навыков

Технологический стек:
- Python 3.12
- aiogram 3.x
- Pydantic Settings
- SQLite + SQLAlchemy
- ChromaDB для векторной памяти
- OpenRouter для LLM
- Docker & Docker Compose

## Полезные команды

```bash
# Запуск бота
docker compose up -d --build

# Просмотр логов
docker compose logs -f

# Перезапуск
docker compose restart

# Остановка
docker compose down
```

## Структура проекта

```
Alex-Nano-Bot/
├── app/
│   ├── agents/          # AI агенты
│   ├── core/            # Ядро (конфиг, БД, память)
│   ├── handlers/        # Обработчики Telegram
│   ├── utils/           # Утилиты
│   └── bot.py          # Точка входа
├── skills/              # Навыки бота
├── data/               # База данных
└── docker-compose.yml
```

## Конфигурация

Файл `.env` должен содержать:
```env
BOT_TOKEN=your_telegram_token
OPENROUTER_API_KEY=your_openrouter_key
ADMIN_IDS=123456789
```

**ВАЖНО:** ADMIN_IDS теперь поддерживает:
- Один ID: `ADMIN_IDS=123456789`
- Список: `ADMIN_IDS=123456789,987654321`
