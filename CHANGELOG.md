# Changelog

All notable changes to Alex-Nano-Bot will be documented in this file.

## [1.0.0] - 2024

### Added
- Initial release
- Three agent modes: Nanobot, Claudbot, Moltbot
- Vector memory system with ChromaDB
- Dynamic skills loading and creation
- SQLite database for persistence
- Telegram bot interface with aiogram 3.x
- Docker and Docker Compose support
- Multi-model LLM support via OpenRouter
- Memory management (notes, trips, budgets, plans)
- Skill system with code generation
- Conversation history
- Comprehensive logging

### Features
- **Nanobot**: Quick assistant for everyday tasks
- **Claudbot**: Smart planner with multi-step reasoning
- **Moltbot**: Skills manager and catalog
- Vector-based semantic search
- Async/await throughout
- Production-ready Docker setup
